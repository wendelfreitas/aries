## :rocket: **How to Run**

```sh
git clone git@github.com:wendelfreitas/aries.git
cd aries

# You can use yarn, npm or pnpm.
$ pnpm install

# To run project
$ pnpm run dev


# To run storybook and see all components created
$ pnpm run storybook


# To run Jest and check coverage
$ pnpm run test
```
